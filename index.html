O HTML

Kaj je HTML

HTML je kratica za Hyper Text Markup Language (slov. jezik za označevanje nadbesedila). Je označevalni jezik pri katerem se preko značk in atributov izdelujejo spletne strani. Predstavlja osnovo spletnega dokumenta. Z njim se določa samo vsebino in prikaz v spletnem brskalniku, stran pa se oblikuje s pomočjo CSS. HTML se lahko izdela v vsakem urejevalniku besedila (npr. beležnica).

Elementi HTML so sestavljeni iz oznake, zapisanih v špičastih oklepajih ( <element> ). HTML oznake so napisane v parih (npr. <element> vsebina </element> ). Znotraj para lahko oblikovalec piše vsebino ali pa druge oznake, kar se imenuje gnezdenje oznak.

Vsak ima dostop do ogleda HTML kode katere koli javne spletne strani z uporabo bližnjice Ctrl+U

Zgodovina HTML in različice

Leta 1991 je na inštitutu CERN Tim Berners-Lee razvil prvo različico HTML, protokol HTTP in izdelal prvi spletni strežnik. Kmalu po temu, okoli leta 1993, je prišlo do velike rasti interneta. Takrat se je pojavil prvi brskalnik za splošno uporabo, ki ga je podpiral HTML, imenovan Mosaic.

Novembra 1995 je bila pod iniciativo IETF (Internet Engineering Task Force) razvita nova različica HTML, HTML 2.0. Ta je vsebovala vse elemente prejšnje različice, z nekaj dodanih na novo. Bila je standard za razvoj spletnih strani do leta 1997.

Različica HTML 3.0 je predstavljala več težav. V razvoj spletnih strani je bilo vpletenih vse več ljudi, kar je pomenilo tudi več zahtev za nove elemente, ki jih takratni HTML še ni podpiral. Ko je skupina, ki jo je vodil Dave Raggett, razvila HTML 3.0 z veliko novimi elementi, ki so jih ljudje zahtevali, se je pokazalo da takratnji brskalniki še niso bili dovolj zmogljivi za ta napredek. HTML 3.0 se zaradi tega ni nikoli zares uveljavil. Ker pa je nezmogljivost brskalnikov skozi čas predstavljala vse večjo težavo, je W3C (World Wide Web Consortium) v svoji prvi objavi predstavil očiščen HTML, kjer so odstranili večino elementov, ki so jih poskušali uveljaviti ostali.
Leta 1997 je HTML 3.2 postal uradni standard, ki ga vsi brskalniki v celoti podpirajo še danes.

HTML 4.0 je vseboval še več novih elementov, vključno s podporo CSS (Cascading Style Sheets, slovensko prekrivni slogi). HTML 4.1 je bila zadnja klasična verzija HTML, po tem pa se pojavi XML, s čimer je bilo potrebno posodibiti tudi HTML.

XML in XHTML

XML je kratica za eXtensible Markup Language, in je preprost računalniški jezik, podoben HTML, z nekaj razlikami: - XML nima vnaprej določenih oznak, kot jih ima HTML - HTML je bil narejen za prikaz podatkov - XML pa za prenos podatkov

XHTML 1 je bila vmesna stopnja med HTML 4.0 in XML. Ta verzija je združljiva s HTML 4.0 in hkrati podpira vsa pravila XML.

Po tem je prišla verzija HTML 5, ki je do danes najnovejša verzija. Podpira vse, kar je podpirala verzija XHTML 1, ima pa dodanih še precej novosti, kot npr. DOCTYPE (definiranje tipa dokumenta), več možnosti na podričju multimedije (dodajanje avdio in video vsebin brez vtičnikov) ter več novih oznak.
