O CSS

CSS

CSS (Cascading Style Sheets) so prekrivni slogi, ki spletnemu brskalniku povejo, kako naj prikaže HTML dokument. CSS strukturira vsebino HTML dokumenta in mu da želeno obliko. Z uporabo slogov lahko določimo vse od ozadja, barv, pisave, robov, zamikov, poravnave itd.

Zgodovina CSS

V začetni dobi spleta na voljo ni bilo veliko oblikovanja strani po okusu razvijalcev. Določilo se je lahko, kaj določen del predstavlja (npr. slike, vrstice, polja), kar pa je brskalnik nato oblikoval po svoje.

Po tem so se za oblikovanje strani pojavile oznake, ki so omogočale razvijalcem oblikovati strani po njihovem okusu. Težava pa je bila, da je bilo oblikovanje včasih zelo zamudno, ker se je moralo spremeniti vsako stran posebej. Zato so bile razvite slogovne predloge, ki so razvijalcem omogočile hitrejše spreminjanje videza (samo preko datoteke s slogovnimi predlogami).

Verzije CSS

Oktobra 1994 je Håkon Wium poslal svoj predlog pravil CSS organizaciji W3C. Njegov predlog je bil sprejet, in skupaj s članom W3C Bertom Bosom sta izdala CSS. CSS specifikacija se je razglasila za priporočilo decembra 1996, ko je izšel CSS 1.

Maja 1998 je izšla končna verzija CSS 2, ki je bila dopolnjena leta 2002.
CSS 3 pa se je začel razvijati že prej, v letu 1999. Za razliko od ostalih verzij se ta deli na module in bazira na prejšnjih verzijah. CSS 3 je trenutno najnovejša verzija CSS.
